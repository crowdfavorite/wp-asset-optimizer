Asset Optimizer

CrowdFavorite's Asset Optimizer takes all the separate CSS and JS files included in plugins and external add-ons, and compiles them into one file, helping your pages load faster.

| This will automatically recompile all CSS and JS files whenever they change.
| This will stop any files from being compiled.
| This lets you choose what you want compiled and what you want to exclude.

Select this option if your site is using a plugin like WP Super Cache, or another static caching solution, to ensure that the concatenated files are generated and served before the cache occurs.
